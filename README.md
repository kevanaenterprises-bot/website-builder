# AgencyGenius AI Pipeline

Full-stack lead generation and site preview dashboard for local business outreach in Frisco, TX.

## Stack
- React + Vite + TypeScript
- Tailwind CSS + @blinkdotnew/ui
- @tanstack/react-router + @tanstack/react-query
- framer-motion, recharts

## Features
- Lead dashboard with 18 pre-loaded Frisco, TX businesses
- Instant site preview generation per business category
- Sales call scripts with personalized hooks
- Performance analytics with charts
